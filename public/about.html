About This Workshop

This workshop is designed to teach developers how to use GitHub Actions for automating their software development workflows.

Topics Covered

  • Setting up your first workflow
  • Understanding workflow triggers and events
  • Running builds and tests
  • Working with jobs and steps
  • Managing job dependencies

Prerequisites

  • Basic understanding of Git and GitHub
